# How to use?
* Clone or download this repository;
* Install docker and docker-compose;
* Open project;
* Write `docker-compose up -d`

# About services
* backend (php) - localhost:8080
* frontend (react) - localhost:3000
* phpmyadmin (mariadb)- localhost:8081

### For PhpMyAdmin
* server - mariadb
* username - root
* password - root
* database - sandbox
* table - category
